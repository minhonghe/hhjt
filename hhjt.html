公司介绍

鸿鹄网络科技有限公司成立于2025年，总部位于抚州，公司拥有超过50名员工，其中80%为技术研发人员。

我们专注于智慧家庭恒温系统、智慧医疗系统、农业智能恒温系统人工智能、大数据分析、云计算和物联网技术的研发与应用，为全球客户提供创新的数字化解决方案。

公司已获得国家高新技术企业认证，拥有超过100项自主知识产权专利，

我们的使命是通过技术创新解决复杂问题，愿景是成为全球领先的科技解决方案提供商。

API接口

我们提供以下API接口供合作伙伴使用。所有API请求都需要有效的API密钥。

GET
https://api.hhjtpt.com.cn/v1/company/info

获取公司基本信息，包括公司介绍、联系方式等。

GET
https://api.hhjtpt.com.cn/v1/products

获取产品列表，支持分页和分类筛选。

POST
https://api.hhjtpt.com.cn/v1/contact

提交联系请求，需要验证码验证。

API使用示例

// 获取公司信息
fetch('https://api.jiangxishengfuzhoushi.com.cn/v1/company/info', {
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY'
    }
})
.then(response => response.json())
.then(data => console.log(data));
